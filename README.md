# tools
## 個人用に作ったコマンドの保管用

### rename  

ファイルの名前を「任意の名前」＋「連番」に変更するシェルスクリプト  

動作  
XXXX.jpg →　任意の名前-000.jpg  

### google  

w3mを使ってgoogle検索をするシェルスクリプト  

使用例  
google 検索ワード  

_注　w3mが要ります_  

### tenki  

curlを使って指定された場所の天気予報を表示するシェルスクリプト  

使用法  
tenki tokyo

_注　curlが要ります_  
_デフォルトは広島です_  

### rename.go  

Go言語で作ったファイル名を「任意の名前」＋「連番」に変更するコマンド  
例  
XXX.jpg →　任意の名前-000.jpg
