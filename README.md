# tools
## 個人用に作ったコマンドの保管用

rename<br>
<br>
ファイルの名前を任意のものに変更するシェルスクリプト<br>
<br>
動作<br>
XXXX.jpg →　任意の名前-000.jpg<br>
<br>
google<br>
<br>
w3mを使ってgoogle検索をするシェルスクリプト<br>
<br>
使用例<br>
google 検索ワード<br>
注　w3mが要ります<br>
<br>
tenki<br>
<br>
curlを使って指定された場所の天気予報を表示するシェルスクリプト<br>
<br>
tenki tokyo<br>
注　curlが要ります<br>
　　デフォルトは広島です<br>
